# LangGraph

🧠 LangGraph Agents – Practical Implementation

This project demonstrates hands-on implementation of agentic workflows using LangGraph and LangChain, focusing on building dynamic, stateful AI systems.

📘 Overview

The goal of this project is to explore how modern AI agents are built using graph-based execution instead of traditional linear pipelines.

It includes implementations of:

Looping agents

Conditional agents

Sequential workflows

Multi-input handling

Basic tool-augmented agents

This project serves as a foundation for building advanced agentic AI systems and RAG pipelines.

⚙️ Tech Stack

Python

LangGraph

LangChain

Ollama

Jupyter Notebook / VS Code
